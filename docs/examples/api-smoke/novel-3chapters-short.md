# 第一章 雨夜来信

晚上十一点四十五分，林舟离开公司时，整栋写字楼只剩安全出口的绿灯还亮着。雨水沿着玻璃幕墙往下流，像有人在夜色里反复擦掉城市的轮廓。

他刚走到一楼大厅，保安递给他一封没有署名的信。信封很旧，封口处贴着一枚褪色的邮票，里面只有一张被雨水洇开的纸条：如果还想找到你姐姐，明晚去青石巷的旧书店。

林舟站在旋转门前，手指慢慢攥紧纸条。三年前，姐姐林夏就是在一个雨夜失踪的。警方早已结案，可这封信让所有被压下去的疑问重新浮出水面。

# 第二章 旧书店

第二天傍晚，林舟来到青石巷。旧书店夹在两家关门的杂货铺之间，招牌只剩一半，门口挂着一盏昏黄的灯。

店主没有问他来意，只把一本《城市旧闻》推到柜台边。书页中夹着一张照片，照片上是姐姐林夏和一个陌生男人站在旧钟楼前。照片背面写着：钟声停下时，不要相信第一个开门的人。

林舟追问店主陌生男人是谁，店主却摇头说，三年前林夏也问过同样的问题。窗外忽然传来钟声，明明旧钟楼已经废弃多年。

# 第三章 街灯

林舟沿着钟声走到旧城区尽头。街灯一盏接一盏闪烁，像有人在黑暗里给他指路。尽头是一扇锈迹斑斑的铁门，门缝里透出微弱的光。

铁门后是废弃的钟楼大厅，墙上贴满剪报，全都和多年前的失踪案有关。林舟在最中央的桌上看见姐姐留下的录音笔。

录音笔里传出林夏的声音：如果你听到这里，说明他们又开始行动了。不要回头，去找许岚，她知道名单上的最后一个名字。
