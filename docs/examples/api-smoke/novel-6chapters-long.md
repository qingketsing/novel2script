# 第一章 雾中灯塔

海雾封住港口的第三个夜晚，苏棠收到一只没有寄件人的木盒。盒中只有一把生锈的铜钥匙和半张灯塔值班表，纸角写着哥哥苏远失踪前留下的一句话：午夜前，让旧灯再亮一次。

苏远曾是港口灯塔的维护员，三个月前在一次风暴中失踪。官方结论是意外落海，可苏棠始终不信。她带着木盒去码头修理铺找陈默，那个沉默的修理师曾和苏远一起维护旧灯塔。

陈默认出钥匙属于已经停用五年的地下机房。他提醒苏棠，旧灯塔早被新导航系统取代，贸然靠近可能违反港口禁令。可窗外雾笛突然响起，远处黑暗中的灯塔闪过一道微弱白光。

苏棠望着光说，如果哥哥还活着，他一定在等这个信号。陈默沉默片刻，拿起工具箱，决定陪她走一趟。

# 第二章 潮汐密道

退潮后，苏棠和陈默沿礁石走到灯塔背面。铜钥匙打开了一扇被海草覆盖的铁门，门后是一条向下延伸的狭窄通道。墙上留着苏远画下的潮汐刻度，每一道刻痕都指向午夜。

地下机房里，备用发电机还有燃料，但连接主灯的电缆被人为切断。陈默检查断口，发现切割痕迹很新，绝不是自然老化。苏棠在工具柜中找到一台录音机，里面有苏远急促的声音。

录音里说，如果主灯熄灭，货船会被引向北礁。有人利用浓雾和伪造导航制造事故，事故发生后，码头仓库里的走私货物就会被悄悄转移。

通道深处传来铁门合拢的回声。海水开始漫过入口，留给他们的时间不多了。

# 第三章 仓库名单

两人暂时离开灯塔，潜入码头旧仓库寻找录音中提到的证据。仓库门口贴着停用封条，里面却有近期搬运过的痕迹，地面上残留着新鲜轮胎印。

苏棠在一只铁皮柜里找到半本航线记录，记录上有多个被涂黑的船名。陈默用修理灯照着纸页，发现涂黑处下方仍能看见几个字母，对应的正是即将在午夜进港的货船“白鸥号”。

仓库二层突然传来脚步声。苏棠和陈默躲进货架间，看见港务处副主任罗峻带着两名陌生人进来。他们谈到旧灯塔、伪造信号和苏远留下的麻烦，语气像是在处理一件早该结束的事。

苏棠忍住冲出去质问的冲动，用手机拍下他们交换文件的画面。就在这时，手机低电量提示音响起，仓库里的所有人都停了下来。

# 第四章 断电的港口

罗峻的人封锁了仓库出口。陈默带着苏棠从维修管道逃出，却发现整片港口突然断电，只有新导航塔的红灯还在雾中闪烁。广播系统传来故障提示，所有船只被要求按自动信号进港。

陈默判断，新导航塔已经被人篡改，白鸥号会被引向北礁。要阻止事故，只能在午夜前重启旧灯塔主灯，用人工灯号覆盖错误信号。

苏棠终于问陈默，为什么他比任何人都了解旧灯塔。陈默承认，当年他和苏远一起发现走私线索，但因为害怕丢掉工作，他选择沉默。苏远失踪后，他一直等一个补救的机会。

苏棠没有责怪他，只把铜钥匙递回他手里。她说，等不是补救，行动才是。

# 第五章 午夜重燃

距离午夜只剩十分钟。陈默启动发电机，苏棠带着修好的电缆爬上灯塔。狂风推着海雾撞向玻璃，主灯的转轴因为锈蚀停在原位。

港外，白鸥号正按照错误的引导灯驶向北礁。罗峻带人赶到灯塔下，试图切断发电机。陈默守在机房，用扳手卡住铁门，为苏棠争取最后几分钟。

苏棠在塔顶接通电缆，却发现主灯仍然无法旋转。她想起哥哥录音里反复提到的潮汐刻度，终于明白那不是时间，而是转轴校准角度。她用尽力气拨动转轴，第一束灯光穿透雾层。

白鸥号拉响汽笛，缓慢偏离北礁。灯塔下的罗峻脸色惨白，警笛声从码头方向逼近。

# 第六章 黎明证词

天亮前，海警船靠上码头。苏棠把手机里的视频、航线记录和苏远的录音交给调查员。罗峻被带走时，仍然辩称自己只是服从更高层的安排。

旧灯塔控制台下弹出一个暗格，里面放着苏远整理的完整证据和一封写给苏棠的信。信里说，如果她看到这些，说明自己没能按时回来，但他相信她会让灯重新亮起。

苏棠站在塔顶，看见雾一点点散开。陈默坐在台阶上，手臂受了伤，却第一次露出轻松的笑。港口广播恢复正常，白鸥号平安靠岸。

苏棠把哥哥的值班表夹进灯塔日志。她知道有些人不会回来，但有些光，只要有人愿意守住，就不会真正熄灭。
