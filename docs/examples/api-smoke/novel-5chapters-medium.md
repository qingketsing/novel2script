# 第一章 没有终点的车票

晚上十一点四十五分，林川离开公司时，雨正落在城市的高架桥上。末班公交停在路边，车灯把积水照成一片发白的光。

司机没有说话，只递给他一张没有终点站的车票。林川以为是售票系统故障，直到车门关闭后，他发现车厢里还坐着三个人：抱着红书包的小女孩、灰衣女人，以及一个一直低头看表的老人。

公交车驶过熟悉的路口，却没有停站。窗外的建筑逐渐变得陌生，广播里传出沙哑的提示音：下一站，十年前。

# 第二章 消失的第四位乘客

车驶入一条林川从未见过的隧道。灯光忽明忽暗，每闪一次，车厢里就会少一点声音。

老人抬头告诉林川，这辆车每晚只载四个有遗憾的人，能下车的人只有一个。话音刚落，老人座位上只剩下一块旧怀表，表盘停在零点。

灰衣女人让林川不要碰怀表，可红书包女孩却把怀表塞进他的掌心。表盖里夹着一张泛黄照片，照片上是十年前桥上的车祸现场。

# 第三章 红书包里的照片

小女孩一直没有说话。她抱着红书包，鞋尖一下一下碰着车厢地板。林川坐到她对面，看见书包拉链上挂着一枚熟悉的钥匙扣。

那是他父亲生前送给他的东西。十年前的雨夜，父亲在大桥车祸中去世，而林川一直记得自己因为加班没有接到最后一通电话。

小女孩终于开口，说照片里少了一个人。林川低头再看，发现照片边缘站着灰衣女人，而她的手里拿着一张写有林川名字的车票。

# 第四章 灰衣女人的秘密

车厢里只剩林川和灰衣女人。隧道墙壁上的数字从十倒数到三，每经过一个数字，车窗外就闪过一段旧日记忆。

灰衣女人承认自己叫许岚，是十年前那场事故的幸存者。她当年为了逃避责任，隐瞒了公交刹车被人为破坏的证据。

她说林川的父亲不是意外身亡，而是为了让满车乘客活下来，强行把车转向了桥边护栏。林川握着照片，第一次明白自己这些年的恨找错了方向。

# 第五章 第三个选择

公交车停在大桥中央。车门打开，外面是十年前的暴雨。桥边站着林川的父亲，仍然穿着那件旧夹克。

许岚告诉林川，他可以选择留下来改变过去，也可以带着证据回到现在，让真相被公开。红书包女孩站在车门边，把钥匙扣还给他。

林川走下车，又停住。他没有奔向父亲，而是把车票交给许岚：这一次，我们一起把真相带回去。零点钟声响起，公交车重新驶向城市的灯火。
