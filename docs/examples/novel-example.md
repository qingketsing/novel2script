# 第一章 雾中的灯

海雾封住港口的第三个夜晚，苏棠收到一只没有寄件人的木盒。盒中只有一把生锈的铜钥匙和半张灯塔值班表，纸角写着哥哥苏远失踪前留下的一句话：“午夜前，让旧灯再亮一次。”

苏棠赶到码头修理铺，请陈默辨认钥匙。陈默曾负责维护旧灯塔，他认出钥匙属于已经停用五年的地下机房。窗外的雾笛突然响起，远处黑暗中的灯塔闪过一道微弱白光。

“那里早就断电了。”陈默放下钥匙。

苏棠望着灯塔说：“所以我更要去看看。”

## 第二章 潮汐密道

退潮后，苏棠和陈默沿礁石走到灯塔背面。铜钥匙打开了一扇被海草覆盖的铁门，门后是一条向下延伸的狭窄通道。墙上留着苏远画下的潮汐刻度，每一道刻痕都指向午夜。

地下机房里，备用发电机仍有燃料，但连接主灯的电缆被人为切断。陈默修复线路时，苏棠在工具柜中找到苏远的录音机。

录音里传来急促的声音：“如果主灯熄灭，货船会被引向北礁。有人在利用浓雾制造事故。”

通道深处传来铁门合拢的回声。海水开始漫过入口，留给他们的时间不多了。

## 第三章 午夜重燃

距离午夜只剩十分钟。陈默启动发电机，苏棠带着修好的电缆爬上灯塔。狂风推着海雾撞向玻璃，主灯的转轴因为锈蚀停在原位。

港外，一艘货船正按照错误的引导灯驶向北礁。苏棠用扳手撬动转轴，陈默在机房重新接通电源。第一束灯光穿透雾层时，货船拉响汽笛并缓慢转向。

灯塔控制台下弹出一个暗格，里面放着苏远收集的航线记录和事故证据。苏棠终于明白，哥哥留下的线索不是告别，而是一场等待完成的求救。

天亮前，海警船靠上码头。苏棠把记录交给调查员，回头看见旧灯仍在雾中稳定旋转。
